This project is a replica created for a technical test. It's built with SvelteKit, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository
2. Install dependencies:

```sh
npm install
```

## Running Commands

### Development

Start the development server:

```sh
npm run dev
```

Or start the server and open the app in a new browser tab:

```sh
npm run dev -- --open
```
